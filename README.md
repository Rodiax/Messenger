# Messenger
Chat app built on websocket protocol.

## Configuration
- All you need is at least 2 PC connected to the same local network with installed nodejs server.
- Install nodejs server on PC and run `node app.js` command.
- Change the IP address in `client/client.js` file on line 4. to server address that is running `app.js` file. To get server IP address on Windows just open command prompt and type `ipconfig` basicly it starts with 192.X.X.X address.
- Open `client/index.html` file in browser and start typing.
